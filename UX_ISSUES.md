# UX 이슈 메모

시안·이미지를 **우선 그대로 구현**한 뒤, UX상 이상하거나 개선이 필요한 것만 여기에 적습니다.  
구현을 막지 않고, 나중에 따로 손볼 목록입니다.

## 규칙

1. 먼저 시안/이미지 기준으로 만든다.
2. 너무 이상한 점만 여기 기록한다. (사소한 취향은 적지 않음)
3. 기록 형식: 화면 ID · 현상 · 왜 이상한지 · 제안(선택)

---

## 미해결

### D2 마음 기록 · 감정 에셋 파일명
- **현상:** ~~한글 파일명 오타(`걱정되요.svg`, `붎ㄴ해요.svg`)~~ → 영문 정리됨 (`mood-worried.svg`, `mood-uncomfortable.svg` 등)
- **이유:** 라벨·에셋 이름이 어긋나면 교체·검색이 어려움 · Hangul 경로가 Metro require에 취약
- **현재 구현:** `moodEmojiXml.ts` 인라인 SVG + 에셋은 `mood-*.svg` 영문 파일명
- **제안:** 완료 — 추가 조치 불필요

### D3 마음일기장 · 기록 개수
- **현상:** 시안은 「총 17개 기록」, 데모 데이터는 해당 월 5건
- **이유:** 실제 저장 연동 전 샘플만 유지
- **제안:** 저장 연동 시 실제 카운트 노출

### D5 마음을 담았어요 · 펫 반응
- **현상:** 시안은 「펫 반응」플레이스홀더, 구현은 기존 `DogExpr.wink` 표시
- **이유:** 전용 리액션 에셋 없음 · 플레이스홀더 문구보다 기존 펫이 자연스러움
- **제안:** 전용 리액션 에셋이 오면 교체

### A10 코드 분실 안내 시트 (기록 이어보기)
- **현상:** 시안 배경이 「마음일기 기록」 목록으로 깔려 있음
- **이유:** 이어보기 플로우와 무관한 화면이 뒤에 보이면 맥락이 깨짐
- **현재 구현:** 분실 팁 화면(`lost`) 위에 바텀시트만 올림
- **제안:** 시안 배경을 이어보기/분실 화면으로 교체하거나, 딤만 쓰는 오버레이로 통일

### cm-01-welcome (펫 홈 첫걸음 시트)
- **현상:** 시안 배경의 「다시 만나서…」헤더는 탭 복귀 인터랙션으로 구현됨. 시트와 동시에 뜨면 문구가 겹칠 수 있음
- **제안:** 시트 중에는 탭 환영 헤더를 숨기거나, 시트 카피와 분리

### cm-03 대화 투어
- **현상:** 투어 카드가 펫·말풍선을 많이 가림
- **이유:** 카드가 중앙에 크게 떠 있고 스크림이 캐릭터 위를 덮음
- **제안:** 카드 위치를 입력창 바로 위로 고정하거나, 하이라이트만 남기고 카드 높이 축소

### cm-04 마음일기 투어
- **현상:** 시안 헤더는 타이틀+햄버거만인데, 현재는 부제·일기장 아이콘
- **이유:** 일기 메인 헤더가 아직 D1 피그마 맞춤 전
- **제안:** D1 맞춤 때 헤더도 같이 정리

### cm-05 마음챙김 투어
- **현상:** 시안은 안내 배너+평가도구 리스트인데, 현재 살피기 탭은 데일리체크·점수 카드가 먼저 있음 → 투어 중 스포트라이트 카드로 평가도구를 별도 표시
- **이유:** E1 레이아웃이 시안과 다름
- **제안:** E1 맞춤 시 리스트를 시안처럼 올리고, 실물 행을 하이라이트하도록 바꾸기

### cm-05 마지막 스텝 버튼
- **현상:** 시안·구현 모두 4/4에서도 「다음」
- **이유:** 마지막인데 다음이라 종료 느낌이 약함
- **제안:** 「시작하기」/「완료」로 바꾸거나 스킵 없이 종료 카피 검토

### 최초 진입 이용 안내 팁
- **현상:** _(헤더는 시간대 상태 문구로 맞춤됨)_
- **닫힘:** 메인 헤더를 시안 인사/상태 문구로 통일

### 최초 진입 팁 · 코치마크 순서
- **현상:** 코치마크(cm-01~05) 중에는 팁을 숨기고, 스킵/투어 종료 후 노출
- **이유:** 동시에 뜨면 안내가 과함
- **제안:** 시안 플로우가 「팁 먼저」면 순서 교체

### 스트레스 평가 · 구간 라벨
- **현상:** 구간 카드는 「중등도 이상의 스트레스」(11–20), 최근 결과 shortLabel은 「경도 수준」
- **이유:** 시안 그대로
- **제안:** 결과 라벨을 「중등도 이상」으로 통일할지 디자인 확인

## 반영 완료

- 펫 홈 헤더: 닉네임 → 시간대 상태 문구 (`좋은 아침이에요!` / `오늘 하루 바쁘게…` / `오늘 하루도 정말 수고 많았어요.`)
- 받기 메뉴: 숫자 뱃지 → 노란 글로우 + 「제작 완료」 / 수령 후 빨간 쿨다운 `HH:MM:SS`
- 펫 이름 옆 연필 아이콘
- 최초 진입 이용 안내 팁 (받기·보관)
- 탭 복귀: 「다시 만나서 반가워요!」+ 부제 · 말풍선 「오늘도 화이팅해요」
- 캐릭터 탭: 말풍선 순환 (`오늘 뭐해요?` → `배고파요` → …)
- 사료 주기: 「밥 먹어서 에너지가 올랐다멍/냥!」+ 하트 6개 버스트
- 놀아 주기: 「놀아줘서 에너지가 늘었다멍/냥!」+ 하트 버스트 · 펫별 이미지
